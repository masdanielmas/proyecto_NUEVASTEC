# Proyecto NUEVASTEC

Este proyecto corresponde a la API para la plataforma educativa **NuevasTec**, desarrollada con **Node.js**, **Express**, y **MongoDB**.

## Estructura general
- **models/**: Contiene los modelos Mongoose de la base de datos.
- **routes/**: Define las rutas principales de la API (usuarios, cursos, progreso, etc.).
- **middleware/**: Incluye las funciones de autenticación JWT.
